# Agent Skills & Meta-Skills Repository

A centralized repository for maintaining agent skills and meta-skills for software development tasks.

## Structure

```
agent_skills/
├── skills/           # Individual agent skills
├── meta-skills/      # Composite skills that combine multiple skills
└── docs/            # Documentation
```

## Creating Skills

Use the `skill-creator` skill to generate new skills following the standard skill format.
